# vimrc
My .vimrc file

To Set Up:

```cd ~;```

```git clone https://github.com/tuckerww/vimrc.git```

```mv vimrc/.vimrc .```

```rm -rf vimrc```

```mkdir -p ~/.vim/bundle ~/.vim/backup ~/.vim/swap ~/.vim/undo```

```git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim```

In Vim:

```:PluginInstall```
